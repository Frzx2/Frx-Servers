# Frx Servers (js version)

> A simple and clean UI to host Minecraft servers locally without port forwarding.

Frx Servers is an open-source server manager that helps you set up Minecraft servers easily on your computer.  
It supports multiple server types and uses Playit.gg tunneling to make your server joinable from anywhere.

---

### Features

- Host Minecraft servers with Playit.gg
- Supports:
  - Vanilla
  - Paper
  - Forge
  - Fabric
- Auto-download and setup of server files
- Detects if Java is missing and helps install it
- Live server console with command input
- CPU / RAM / live server monitoring
- Edit server.properties in a simple UI
- User-friendly and open-source for full modification

---

### Requirements

- Windows 10 or Windows 11
- Java 17+ recommended (if u dont have java the app will  help u download)  
- Playit.gg Agent setup before launching  
  > Download: https://playit.gg/
